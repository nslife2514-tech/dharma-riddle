📸 วิธีใส่รูปประกอบปริศนาธรรม
==============================

โฟลเดอร์ "photos" นี้ใช้เก็บรูปประกอบของปริศนาแต่ละข้อ
รูปจะปรากฏในหน้า "เฉลย" หลังจากผู้เล่นตอบถูก


🎯 ต้องการรูปทั้งหมด 11 รูป — ตั้งชื่อตามนี้เป๊ะๆ:
──────────────────────────────────────────────

  01-cage.jpg       ← รูปกรงนกที่เปิดอยู่
  02-log.jpg        ← รูปขอนไม้ลอยน้ำในบ่อ
  03-bell.jpg       ← รูประฆังพร้อมโซ่ดึง
  04-rock.jpg       ← รูปก้อนหินทับหญ้า
  05-broom.jpg      ← รูปไม้กวาดกับใบไม้แห้ง
  06-dolls.jpg      ← รูปตุ๊กตาปูน
  07-fountain.jpg   ← รูปอ่างน้ำพุ
  08-wall.jpg       ← รูปกำแพงก้อนหิน
  09-lamp.jpg       ← รูปตะเกียงเปล่า
  10-well.jpg       ← รูปบ่อน้ำ
  11-rose.jpg       ← รูปดอกกุหลาบบนกำแพง


✅ วิธีเซฟรูปจากแชท Claude:
──────────────────────────────────────────────

  วิธีที่ 1 — บน Mac/PC (ง่ายที่สุด):
    1. เลื่อนกลับไปหารูปที่ส่งให้ Claude ในแชท
    2. คลิกขวาที่รูป → "Save Image As..." (บันทึกเป็น)
    3. ไปที่โฟลเดอร์ Automatic System →
       02-dharma-riddle-ตามหาปริศนาธรรม → photos
    4. พิมพ์ชื่อไฟล์ใหม่ (เช่น 01-cage)
       ตรวจให้แน่ใจว่านามสกุลคือ .jpg
    5. กด Save
    6. ทำซ้ำกับรูปอีก 2 รูป

  วิธีที่ 2 — บนมือถือ:
    1. กดค้างที่รูปในแชท → Save image
    2. รูปจะไปอยู่ในแกลเลอรี่
    3. เปิดโปรแกรมจัดการไฟล์ (Files / My Files)
    4. ย้ายรูปไปที่โฟลเดอร์ photos นี้
    5. เปลี่ยนชื่อให้ตรง


⚠️ ข้อควรระวัง:
──────────────────────────────────────────────

• ชื่อต้องเป็นภาษาอังกฤษเป๊ะ — ตัวพิมพ์เล็ก
• ต้องลงท้ายด้วย .jpg (ห้ามเป็น .jpeg .JPG .png)
• ห้ามมีเว้นวรรคในชื่อ
• ถ้ารูปเป็น .png ให้ใช้โปรแกรม preview เซฟเป็น .jpg
  (หรือบอก Claude ให้แปลงให้)


💡 ถ้าตั้งชื่อผิด:
──────────────────────────────────────────────

เกมจะไม่พัง — หน้าเฉลยจะแค่ไม่แสดงรูป (ซ่อนอัตโนมัติ)
ลองแก้ชื่อใหม่ แล้วเปิดเกมใหม่อีกครั้ง

──────────────────────────────────────────────
